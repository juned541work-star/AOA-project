# AOA-project

## Amplitude of Accommodation Calculator

This project includes a web-based calculator for Hofstetter's Amplitude of Accommodation (AOA) formulas.

### Features
- Enter an age from 0 to 80
- Calculate the minimum, average, and maximum AOA
- Works on desktop and mobile devices
- Ready to deploy as a static website

### Files
- `index.html` — calculator UI
- `style.css` — page styling
- `script.js` — calculation logic
- `server.py` — local static file server for development

### Run locally
1. Open `index.html` in your browser, or
2. Run `python server.py` and open `http://localhost:8000`

### Publish to the web
This repository is configured for GitHub Pages deployment from the `main` branch root. After pushing your changes, GitHub Actions will build and deploy the site automatically.

Expected public URL:
- `https://juned541work-star.github.io/AOA-project/`

If the site does not appear immediately, allow a few minutes for GitHub Pages to finish the deploy.
